# ── Required: Database Engine ─────────────────────────────
#
# MySQL 8.0 or higher
# Download: https://dev.mysql.com/downloads/mysql/
#
# MySQL 8.0+ is mandatory for:
#   - Window functions      → RANK(), LAG(), NTILE(), ROW_NUMBER()
#   - Generated columns     → line_total (GENERATED ALWAYS AS ... STORED)
#   - GROUP BY ROLLUP       → Report 12 (category subtotals)
#   - GROUPING()            → Report 12 (grand total label logic)
#   - CTE (WITH clause)     → Used in all 12 reports
#   - ENUM column types     → order status, return reason, discount type


# ── Recommended: GUI & ERD Tool ───────────────────────────
#
# MySQL Workbench 8.0+
# Download: https://dev.mysql.com/downloads/workbench/
#
# Required to open : erd.mwb (Entity Relationship Diagram)
# Also useful for  : running week2_query.sql, inspecting results
